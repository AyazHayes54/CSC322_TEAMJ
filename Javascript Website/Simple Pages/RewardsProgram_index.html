Rewards Program

Valued customers will be given greater privileges toward the marketplace/checkout sections of PC Emporium.

Ways a customer can gain access to the Rewards Program:

1. Spending $2000 on marketplace items.

2. Spending $1500 on marketplace items. Leaving a thorough review of each verified purchase in the Forum. A thorough review has a post length of 2000+ characters.

3. Spending $1000 on marketplace items. Leaving a thorough review of each verified purchase in the Forum. Sending a formal complaint that was successfully confirmed by the Store Manager.

4. Paying a one-time fee of $250.

Actual details of the Rewards Program:

1. One time only, the user can submit a price match for a specific product. This form would just 2 text boxes, one for productID and one for a link to an alternate seller. If the Store Clerk can confirm that the link shows a different website selling the same item but at a lower price, the price of the item in the Marketplace will be updated to match that website. Otherwise, the price match will be denied and the Store Clerk can send one of two messages back to the user: ‘Invalid Website’ or ‘The item from the alternate seller is not the same as the specified Marketplace item’. Even if denied, the one-time price match privilege is used up.

2. Get 10% off 5 products each month on checkout.

3. Refund privileges. If the user is not satisfied with a purchase, send the item back to the Marketplace. Once received, the user would see a balance increase by the price of that item. Proof of this refund would be through a receipt to the user.

4. At spontaneous times, the user will receive a multiple choice question about future items the website should add to the Marketplace. The user can show preference toward specific items or manufacturers. The Store Clerk/Manager would take this input into account for future decisions.

5. Access to customer service.
